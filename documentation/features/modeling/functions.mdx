  Your [number format](/documentation/getting-started/account/preferences) in Francis determines whether you'll use commas (US) or semicolons (EU) to separate function parameters. All examples below use the US format with commas.

<AccordionGroup>
  <Accordion title='avg("row_name"[range_start:range_end])' defaultOpen="false">
    The `avg()` function calculates the average of the values in the specified time series from `range_start` to `range_end`.

    ```typescript
    // Example
    avg("Revenue"[-3:0])
    ```
  </Accordion>
  <Accordion title='avg_ytd("row_name")' defaultOpen="false">
    The `avg_ytd()` function returns the average of the values in a time series from the beginning of the fiscal year until and including the current month.

    ```typescript
    // Example 
    avg_ytd("Revenue")
    ```
  </Accordion>
  <Accordion title='avg_last_year("row_name")' defaultOpen="false">
    The `avg_last_year()` function returns the average of the values in the previous fiscal year. This function is influenced by your fiscal year settings.

    ```typescript
    // Example
    avg_last_year("Revenue")
    ```
  </Accordion>
  <Accordion title='avg_last("row_name",num_months)' defaultOpen="false">
    The `avg_last()` function returns the average of the values in the time series going back _n_ months, as defined by the `num_months` parameter.

    ```typescript
    // Example
    avg_last("Revenue",6)
    ```
  </Accordion>

  <Accordion title='receivables("row_name", payment_days)' defaultOpen="false">
    The `receivables()` function accumulates revenue amounts from previous months based on an assumption of payment days. Based on the number of payment days, the function loops back through previous months to collect revenue as part of receivables.

    The function assumes 30 days per month. The payment days input must be hardcoded and cannot reference a cell.

    ```typescript
    // Example
    receivables("Revenue", 30)
    ```
  </Accordion>
  <Accordion title="ignore_div_zero(a/b)" defaultOpen="false">
    The`ignore_div_zero()` returns a zero if the function's output is a #DIV\!/0 error. The function is used for margin and percentage calculations, where the denominator is occasionally zero.

    ```typescript
    // Example
    ignore_div_zero("Direct cost"[0] / "Revenue"[0])
    ```
  </Accordion>
  <Accordion title="power(num, exponent)" defaultOpen="false">
    The `power()` function is a mathematical function that returns the value of the parameter _num_ to the power of the _exponent_ parameter.

    ```typescript
    // Example
    power(10,2)
    ```
  </Accordion>

  <Accordion title="round(num, decimals)" defaultOpen="false">

    ```typescript

    ```
  </Accordion>

  <Accordion title="round_up(num)" defaultOpen="false">
    The `round_up()` function rounds numbers _up_ to the nearest integer.

  </Accordion>
  <Accordion title="if(condition, value_if_true, value_if_false)" defaultOpen="false">
    The `if()` function allows you to create a conditional calculation, similar to what is available in Excel or Google Sheets.

    ```typescript
    // Example
    if("Revenue"[0]=0,0,"Gross profit"[0]/"Revenue"[0])
    ```

    If revenue is zero, display zero; otherwise, calculate the gross profit margin. This specific example prevents #ERRs from dividing by zero.
  </Accordion>
  <Accordion title="if_month(a,b)" defaultOpen="false">
    The `if_month` function returns `1` if the time period is equal to the month number input in the formula, else `0`.

    - `if_month(1)` returns `1` if the month is January, else `0`
    - `if_month(1,6)` returns `1` if the month is January or June, else `0`
    - `if_month(1,6,12)` returns `1` if the month is January, June, or December, else `0`
  </Accordion>
  <Accordion title="if_quarter_month(num)" defaultOpen="false">
    The `if_quarter_month` function returns `1` if the time period is equal to the month number inside a given quarter. E.g., you can input [1;3] into the formula.

    - `if_quarter_month(1)` returns `1` if the month is January, April, July, or October
    - `if_quarter_month(2)` returns `1` if the month is February, May, August, or November
    - `if_quarter_month(3)` returns `1` if the month is March, June, September, or December
  </Accordion>
  <Accordion title="weekdays()">
    The `weekdays()` functions returns the numbers of weekdays (Monday through Friday) in a given month.

    ```typescript
    // Example
    weekdays()
    ```
  </Accordion>
